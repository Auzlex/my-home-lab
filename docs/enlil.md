## ENLIL – Network Services Node (Pi-hole)

* **Hostname:** ENLIL
* **IP Address:** 192.168.1.125 **(STATIC - Planned)**
* **Architecture:** ARMv6 (Raspberry Pi 1 Model B)

### Purpose

**ENLIL** will act as a dedicated **network services node** within the home-lab infrastructure.
It will run Pi-hole, providing network-wide ad blocking, DNS sinkholing, and improved privacy
across all devices in the home lab network.

### Role in the Home Lab System

* Provides network-wide ad blocking and DNS services
* Acts as primary DNS server for the home network
* Hosts web interface for Pi-hole administration
* Improves privacy and reduces network traffic
* Serves as centralized network filtering solution
